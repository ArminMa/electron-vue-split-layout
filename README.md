# electron-vue

To start:

```bash
$ yarn install
```

To develop:

```bash
$ yarn dev
# in a new tab, run electron app
$ yarn app
```

To build for production:

```bash
$ npm run build
```

To lint you code:

```bash
$ npm run lint
```
